# Skills Runtime

As skills de engenharia desta instalação devem ser tratadas como instruções de ambiente, não como arquivos versionados neste repositório.

Use este repositório para manter somente:

- `AGENTS.md`
- `docs/agents/*.md`
- documentação de domínio em `CONTEXT.md`, `docs/adr/` e `docs/`

Se uma skill referenciar uma instrução local adicional, siga o caminho indicado no ambiente antes de criar novos arquivos por conta própria.
