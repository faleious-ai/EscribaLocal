# Issue Tracker

Issues e PRDs deste repositório vivem no GitHub Issues do repositório `faleious-ai/EscribaLocal`.

## Convenções

- Criar issue: use `gh issue create`.
- Ler issue: use `gh issue view <numero> --comments`.
- Listar issues: use `gh issue list`.
- Comentar em issue: use `gh issue comment <numero>`.
- Aplicar ou remover labels: use `gh issue edit <numero> --add-label "..."` ou `--remove-label "..."`.
- Fechar issue: use `gh issue close <numero>`.

Quando uma skill pedir para publicar no issue tracker, crie uma issue no GitHub.
Quando uma skill pedir para buscar a ticket relevante, leia a issue no GitHub.
