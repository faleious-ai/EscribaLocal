# Triage Labels

Este repositório usa os cinco rótulos canônicos abaixo, sem mapeamento alternativo.

| Papel canônico | Rótulo no tracker | Significado |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Manter a issue para avaliação |
| `needs-info` | `needs-info` | Aguardando mais informação do autor |
| `ready-for-agent` | `ready-for-agent` | Especificada o suficiente para um agente assumir |
| `ready-for-human` | `ready-for-human` | Requer implementação humana |
| `wontfix` | `wontfix` | Não será tratada |

Use exatamente esses nomes ao aplicar labels de triagem.
